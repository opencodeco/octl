## Getting started

### Installation
Just clone it and link to your `PATH`:
```shell
git clone https://github.com/opencodeco/octl.git ~/octl
ln -s ~/octl/octl /usr/local/bin/octl
```

#### Updates
Then you can just `git pull` for updates:
```shell
cd ~/octl && git pull
```
or
```shell
octl self-update
```
